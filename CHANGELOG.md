# Changelog

## v1.2.0

- パッケージ名と配布exe名を `saga-seeker-html2md` に統一。
- HTML処理の重複パースを削減し、`kuchiki` によるHTML全体のDOM生成を1回に統合。
- 通常コンソール出力を処理開始、完了メッセージ、Enterキー入力待ちのみに最小化。
- `log.txt` に実行環境、対象ファイル、ファイル別結果、エラー詳細、処理時間を記録するように詳細化。
- exe配置ディレクトリを基準に `input` / `output` / `log.txt` を扱う動作を明確化。
- 配布版向けの release profile を追加。
- Windows配布exeに自己署名Authenticode署名を付与する配布手順に整理。

## v1.1.0

- `## キャラクター詳細` 配下の項目を `###` 見出しとして出力するように変更。
- `## スキル` で、スキル名だけでなく説明文も出力するように変更。
- `type` が空の独自スキル・シナリオ報酬スキルも有効なスキルとして扱うように変更。
- 没データの `魅力` をステータス・本文項目として出力しないように調整。
- ステータス抽出時、`魅力` の値が `運` に混ざる問題を修正。
- 内部JSON全体、base64画像、CSS、JavaScript、公式フッターなどの除外処理を維持。

## v1.0.0

- HTMLキャラクターシート / クエストログをMarkdownへ変換する初期版。
- `input` フォルダ内の `.html` / `.htm` を読み取り、`output` に同名 `.md` を出力。
