# セキュリティとデジタル署名について

このツールの配布用実行ファイルには、自己署名のAuthenticode署名を付与します。
ただし、この自己署名は改ざん確認の補助であり、Microsoft Defender SmartScreen の警告を回避するものではありません。
Windowsで初回起動すると「発行元不明のアプリ」や SmartScreen の警告が表示される場合があります。

## 利用者向けの注意

- 配布元を確認したうえで、問題ないと判断できる場合のみ実行してください。
- 受け取ったファイルが改変されていないか確認したい場合は、同梱の `SHA256SUMS.txt` または配布ページに記載されたSHA-256値を確認してください。
- 警告を無条件に無視することは推奨しません。

PowerShellで実行ファイルのSHA-256を確認する例:

```powershell
Get-FileHash .\saga-seeker-html2md.exe -Algorithm SHA256
```

## 開発者向けの注意

正式に「発行元不明」を減らすには、信頼されたコード署名証明書を取得し、`signtool.exe` などで実行ファイルに署名する必要があります。
ただし、署名済みであっても、新しい実行ファイルや配布実績の少ない実行ファイルではSmartScreen警告が表示される場合があります。
利用者に自己署名証明書のインストールは要求しません。
